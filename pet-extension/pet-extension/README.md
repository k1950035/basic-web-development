  - "이름 변경"을 눌러 강아지 이름 변경

  - "새로운 배경 뽑기" 버튼을 누르면 unplash api를 통해 새로운 배경을 불러옴
  - Advice Slip API를 이용하여 5초마다 새로운 명언을 불러옴
